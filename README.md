# 📊✨ Interactive Dashboard (Excel Project)

## 📁 Project Overview
A fully interactive and dynamic Dashboard built using Microsoft 
Excel with slicers, charts, and formulas to explore and 
visualize data in a clean and user-friendly way.

## 🛠 Tools & Technologies Used
- 📊 Microsoft Excel
- 🔄 Pivot Tables & Pivot Charts
- 🎛️ Slicers & Timelines (for interactivity)
- 📐 Excel Formulas (IF, VLOOKUP, SUMIF)
- 🎨 Custom Formatting & UI Design
